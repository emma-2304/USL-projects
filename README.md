# USL Projects

This repository showcases selected academic projects completed for USL coursework, with a focus on practical applications of data analysis and unsupervised machine learning methods.

The collection includes notebook-based projects covering:
- Association Rules
- Clustering
- Dimension Reduction

Each project demonstrates the analytical process from data exploration and methodology to implementation and results interpretation.

## Repository Contents

- `Tam_Nguyen_Bang_(475308)_Clustering_project.ipynb`
- `Tam_Nguyen_Bang_(475308)_Dimension_Reduction.ipynb`

## Purpose
The purpose of this repository is to present coursework projects in a structured and accessible format, highlighting analytical thinking, technical execution, and applied learning.
